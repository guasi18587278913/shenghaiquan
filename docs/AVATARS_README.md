# 头像上传指南

请将需要使用的头像文件放在 `/public/avatars/` 目录下。

## 文件命名规则

- 刘小排的头像：`liuxiaopai.jpg` 或 `liuxiaopai.png`
- 其他用户头像：`user1.jpg`, `user2.jpg` 等

## 上传步骤

1. 将头像文件保存到本地
2. 将文件复制到 `/public/avatars/` 目录
3. 确保文件名正确

## 支持的格式

- JPG/JPEG
- PNG
- WebP

## 建议尺寸

- 最小：100x100 像素
- 推荐：400x400 像素
- 最大：1000x1000 像素
