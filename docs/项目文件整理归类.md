# 深海圈网站项目文件整理归类

### 4. 工具库 (lib/)

## 总结

项目采用Next.js框架，主要功能包括：
1. **社区功能**：动态、成员、消息、通知
2. **学习功能**：课程系统、进度跟踪、排行榜
3. **内容管理**：支持多种CMS集成
4. **用户系统**：完整的认证和用户管理
5. **文件上传**：支持图片和视频上传

项目结构清晰，代码组织合理，文档完善。
