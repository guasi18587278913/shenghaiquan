# 数据导入说明

## 文件放置位置

请将您的学员数据文件放在此目录下：
- 文件名：`students.csv`
- 编码：UTF-8（重要！）
- 格式：CSV格式

## CSV文件要求

1. 第一行必须是列标题
2. 包含以下字段（按照您的原始表格）：
   - 星球成员号
   - 微信昵称
   - 星球昵称
   - 微信ID
   - 星球头像
   - 行业
   - 身份
   - 自我介绍
   - 个人标签
   - 城市
   - 可提供的资源
   - 手机号
   - 微信号
   - 我的标签
   - 你现在的身份

## 导入步骤

1. 将您的CSV文件保存为 `students.csv` 放在这个目录
2. 回到项目根目录运行：
   ```bash
   npm run import-students
   ```

## 注意事项

- 手机号如果为空，会使用微信ID作为登录账号
- 默认密码：deepsea2024
- 重复数据会自动跳过
